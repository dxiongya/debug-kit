# software-cursor.png — attribution

`software-cursor.png` is the "official software cursor (window, 252px)" reference asset
extracted by the **open-computer-use** project (github.com/iFurySt/open-codex-computer-use),
a reverse-engineering of OpenAI Codex Computer Use's on-screen software cursor.

debug-kit renders it as its *virtual pointer* — a purely visual indicator drawn in a
click-through overlay so a human can watch where automated actions land. It is not the
real system cursor and synthesizes no input. Used here for local debugging/visualization.
